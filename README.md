## 🚀 使い方

### 開発環境での起動
```bash
# プロジェクトディレクトリに移動
cd /home/user/webapp

# ビルド
npm run build

# PM2で起動
pm2 start ecosystem.config.cjs

# ログ確認
pm2 logs ai-debate --nostream

# 停止
pm2 delete ai-debate
```

### ユーザー向けの使い方
1. サイトにアクセス
2. 「今すぐ始める」ボタンをクリック（通知が表示されます）
3. 機能セクションで特徴を確認
4. 人気ディベートセクションでサンプルを閲覧
5. 「観戦する」ボタンで近日公開メッセージを確認

### 隠し機能
- **コナミコード**: ↑↑↓↓←→←→BA キーを順番に入力すると、レインボーエフェクトが発動！

## 🛠 技術詳細

### プロジェクト構造
```
webapp/
├── src/
│   ├── index.tsx          # Honoアプリケーション
│   └── renderer.tsx       # レンダラー設定
├── public/
│   └── static/
│       ├── styles.css     # Y2K Gen Zスタイル
│       └── app.js         # インタラクティブ機能
├── ecosystem.config.cjs   # PM2設定
├── wrangler.jsonc         # Cloudflare設定
└── package.json           # 依存関係
```

### 主な依存関係
- **hono**: ^4.11.5（軽量Webフレームワーク）
- **vite**: ^6.4.1（ビルドツール）
- **wrangler**: ^4.4.0（Cloudflare CLI）
- **TailwindCSS**: CDN版（スタイリング）
