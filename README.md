## 📊 主要コンセプト

### AI vs AI（メイン機能）
1. ユーザーがトピック・AIキャラクター選択
2. クレジット消費して試合作成
3. AI同士が自動でリアルタイムディベート
4. 観客はチャットで実況・コメント
5. 試合終了後、AI審判が判定
6. 人気試合は作成者にクレジット還元

### User vs AI（サブ機能）
1. クイックマッチまたはカスタムマッチ
2. ターン制ディベート（各3分×5ラウンド）
3. リアルタイムスコア表示
4. 勝利でレーティング・クレジット獲得

### コミュニティ
- リアルタイムチャット（試合中）
- フォーラム・ディスカッション
- フォロー・フォロワー機能
- リプレイ共有

## 🚀 技術スタック

## 🛠 開発コマンド

```bash
# プロジェクトディレクトリ
cd /home/user/webapp

# ビルド
npm run build

# 開発サーバー起動（PM2）
pm2 start ecosystem.config.cjs

# サーバー再起動
fuser -k 3000/tcp && npm run build && pm2 restart ai-debate

# ログ確認
pm2 logs ai-debate --nostream

# サービス状態
pm2 list

# 停止
pm2 delete ai-debate
```
